# dough-tools

Small web app with calculators for dough hydration, dough temperature, fermentation time, and yeast conversions.

Early side project to make common baking math fast and easy in the browser.

## Planned tools (v1)

- Hydration and baker's percentages (including levain)
- Desired dough temperature / water temperature
- Fermentation time helper (approximate ranges)
- Yeast amount vs time + fresh/active/instant conversions
